# Invernalink 🌱

**An open-source garden monitoring system powered by Home Assistant**

Invernalink is a modular, DIY-friendly system designed to help you monitor and automate your garden, greenhouse, or indoor growing environment. Built with flexibility and openness in mind, it integrates seamlessly with Home Assistant and is designed to be accessible for makers, growers, and developers alike.

## 📦 Repository Contents

This repository includes:

* **Gerber files** – Ready-to-manufacture PCB designs
* **STL files** – 3D-printable enclosures and components
* **YAML configurations** – Home Assistant integration and setup

Everything you need to build, modify, and extend Invernalink is here.

---

## 🚀 Features

* 🌡️ Environmental monitoring (temperature, humidity, CO2, Light etc.)
* 🔌 Home Assistant integration. you can 
* 🧩 Modular hardware design
* 🛠️ Fully customizable. The sensors are meant to be switched out and upgraded.

---

## 🧑‍🔧 Getting Started

1. Download or clone this repository
2. Manufacture the PCB using the provided Gerber files
3. 3D print the enclosure from the STL files
4. Assemble the hardware components
5. Import and adapt the YAML configuration into Home Assistant
6. Start monitoring your garden 🌿

> Detailed build instructions and documentation are available on the project page.
